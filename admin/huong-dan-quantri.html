⛩️ 1. Giới Thiệu Hệ Thống

Hệ thống Quản Trị Nội Dung (CMS) Tâm Linh Việt cho phép các quản trị viên và thành viên cập nhật, chỉnh sửa thông tin về địa điểm đền chùa, lịch lễ hội và hướng dẫn nghi lễ trên website mà không cần kiến thức lập trình.

Cấu trúc hệ thống

den-chua/ ├── admin/ │ ├── login.html ← Trang đăng nhập quản trị │ ├── index.html ← Bảng điều khiển chính │ ├── huong-dan-quantri.html ← Tài liệu này │ ├── css/admin.css │ └── js/admin.js ├── index.html ← Trang chủ website ├── dia-diem.html ← Bản đồ & danh sách đền chùa ├── lich-le-hoi.html ← Lịch lễ hội ├── huong-dan.html ← Hướng dẫn nghi lễ ├── js/ │ ├── map.js ← Dữ liệu địa điểm gốc │ └── calendar.js ← Dữ liệu lễ hội gốc └── css/style.css
ℹ️ Dữ liệu chỉnh sửa qua admin panel được lưu trong localStorage của trình duyệt và tự động hiển thị trên website. Để cập nhật vĩnh viễn vào file JS, dùng chức năng Xuất code JS.

📍 3. Quản Lý Địa Điểm

Phần này cho phép thêm, sửa, xoá thông tin các đền, đình, chùa hiển thị trên bản đồ trang dia-diem.html.

3.1 Thêm Địa Điểm Mới

1
Nhấn 📍 Địa Điểm ở sidebar trái để vào trang quản lý
2
Nhấn nút ➕ Thêm Mới ở góc trên bên phải bảng
3
Điền thông tin vào form:
  • Tên địa điểm – Tên đầy đủ (ví dụ: Đền Ngọc Sơn)
  • Loại – Chọn: Đền/Miếu, Chùa, hoặc Đình
  • Tỉnh/Thành – Chọn trong danh sách 8 tỉnh
  • Địa chỉ – Địa chỉ chi tiết
  • Vĩ độ / Kinh độ – Toạ độ GPS (xem hướng dẫn bên dưới)
  • Giờ mở cửa – Ví dụ: 6:00 – 18:00
  • URL Ảnh – Đường dẫn ảnh trực tiếp
  • Mô tả – Giới thiệu lịch sử, ý nghĩa
4
Nhấn 💾 Lưu Địa Điểm – địa điểm sẽ xuất hiện ngay trên bản đồ

3.2 Sửa Địa Điểm

1
Tìm địa điểm cần sửa trong bảng (dùng ô tìm kiếm hoặc bộ lọc)
2
Nhấn biểu tượng ✏️ ở cột Thao Tác
3
Chỉnh sửa nội dung cần thay đổi trong form
4
Nhấn 💾 Lưu Địa Điểm để lưu thay đổi

3.3 Xoá Địa Điểm

1
Nhấn biểu tượng 🗑️ bên cạnh địa điểm cần xoá
2
Hộp thoại xác nhận xuất hiện – nhấn Xác Nhận để xoá
⚠️ Xoá địa điểm không thể hoàn tác. Hãy chắc chắn trước khi xoá. Nếu cần khôi phục, dùng chức năng Nhập dữ liệu từ file sao lưu.

3.4 Cách Tìm Toạ Độ (Vĩ Độ / Kinh Độ)

1
Mở maps.google.com trong trình duyệt
2
Tìm kiếm tên địa điểm (ví dụ: "Đền Ngọc Sơn Hà Nội")
3
Chuột phải vào đúng vị trí trên bản đồ
4
Menu xuất hiện, nhấn vào dòng toạ độ đầu tiên (ví dụ: 21.0285, 105.8524) để sao chép
5
Dán vào ô Vĩ Độ (số đầu tiên) và Kinh Độ (số thứ hai)
💡 Vĩ độ (Latitude) là số đầu tiên, thường từ 10-23 đối với Việt Nam. Kinh độ (Longitude) là số thứ hai, từ 102-109.

🖼️ 5. Sửa Nội Dung & Hình Ảnh

5.1 Đổi Hình Ảnh Địa Điểm / Lễ Hội

Hệ thống sử dụng URL ảnh trực tiếp từ internet. Cách lấy URL ảnh:

Cách 1: Dùng ảnh từ Unsplash (Miễn phí)

1
Truy cập unsplash.com và tìm kiếm hình phù hợp (ví dụ: "temple Vietnam")
2
Nhấn vào ảnh, nhấn nút Share → Copy link
3
Hoặc nhấn chuột phải vào ảnh → "Sao chép địa chỉ ảnh"
4
Dán URL vào ô URL Ảnh Đại Diện trong form chỉnh sửa

Cách 2: Dùng Google Drive

1
Upload ảnh lên Google Drive
2
Chuột phải vào ảnh → Chia sẻ → Đặt quyền "Bất kỳ ai có đường liên kết"
3
Sao chép link, lấy phần ID (chuỗi ký tự sau /d/)
4
Dùng URL dạng: https://drive.google.com/uc?id=FILE_ID
💡 Kích thước ảnh khuyến nghị: 800×500px trở lên. Định dạng: JPG hoặc WebP. Ảnh quá nhỏ sẽ bị mờ trên màn hình lớn.

5.2 Sửa Nội Dung Nghi Lễ & Văn Khấn

Nội dung nghi lễ được lưu trực tiếp trong file huong-dan.html. Để chỉnh sửa:

1
Mở file huong-dan.html bằng VS Code, Notepad++ hoặc bất kỳ trình soạn thảo văn bản nào
2
Dùng chức năng Tìm kiếm (Ctrl+F) để tìm đoạn cần sửa (ví dụ: tìm "vanKhan1" để sửa văn khấn ông Táo)
3
Chỉnh sửa nội dung văn bản giữa các thẻ HTML (không xoá thẻ)
4
Lưu file (Ctrl+S) và làm mới trang web để kiểm tra
⚠️ Chỉ chỉnh sửa nội dung văn bản giữa các thẻ HTML. Không xoá hoặc thay đổi các thẻ HTML như <div>, <h4>, <button> nếu không có kiến thức về lập trình web.

💾 6. Xuất & Nhập Dữ Liệu

6.1 Sao Lưu Dữ Liệu (Khuyến Nghị Hàng Tuần)

1
Vào menu Xuất / Nhập Dữ Liệu
2
Nhấn 📥 Xuất file JSON (Địa Điểm + Lễ Hội)
3
File JSON sẽ tự tải về máy tính (ví dụ: tamlinhviet-data-2026-03-06.json)
4
Lưu file vào thư mục an toàn, có thể đặt tên theo ngày để dễ quản lý

6.2 Cập Nhật Dữ Liệu Vào File JS (Vĩnh Viễn)

Dữ liệu chỉnh sửa qua admin panel lưu trong localStorage – chỉ có trên máy tính hiện tại. Để cập nhật vào file gốc:

1
Vào Xuất / Nhập Dữ Liệu → nhấn 📄 Xuất code JS
2
Nhấn 📋 Sao Chép để sao chép toàn bộ code
3
Mở file js/map.js bằng VS Code hoặc Notepad++
4
Tìm dòng const LOCATIONS = [, xoá toàn bộ mảng và dán dữ liệu mới vào
5
Làm tương tự với js/calendar.js cho phần const FESTIVALS = [
6
Lưu file và kiểm tra website để xác nhận

🔒 7. Bảo Mật

🔴 Quan trọng: Hệ thống CMS hiện tại là phiên bản tĩnh (static), mật khẩu được lưu trong file JS. Đây phù hợp cho môi trường nội bộ. Nếu triển khai public, cần bổ sung backend bảo mật.

Nguyên tắc bảo mật

  • ✅ Đổi mật khẩu mặc định ngay sau khi triển khai
  • ✅ Không chia sẻ thông tin tài khoản qua email không mã hoá
  • ✅ Đăng xuất sau khi sử dụng xong trên máy tính công cộng
  • ✅ Sao lưu dữ liệu định kỳ (ít nhất 1 lần/tuần)
  • ❌ Không lưu mật khẩu vào trình duyệt trên máy công cộng
  • ❌ Không chia sẻ tài khoản Super Admin

Đổi Mật Khẩu

Mở file admin/login.html bằng trình soạn thảo, tìm đoạn:

const ACCOUNTS = [ { username: 'admin', password: 'TamLinhViet@2026', ... }, { username: 'quantri', password: 'QuanTri@2026', ... }, { username: 'thanhvien',password: 'ThanhVien@2026', ... } ];

Thay đổi giá trị password thành mật khẩu mới. Mật khẩu mạnh nên có ít nhất 12 ký tự, kết hợp chữ hoa, chữ thường, số và ký tự đặc biệt.

❓ 8. Câu Hỏi Thường Gặp

Q: Tôi thêm địa điểm nhưng không thấy trên bản đồ?

Kiểm tra lại toạ độ (vĩ độ/kinh độ). Toạ độ sai sẽ đặt marker ra ngoài vùng nhìn thấy. Toạ độ Việt Nam: Vĩ độ 8–23, Kinh độ 102–110.

Q: Ảnh tôi dán vào không hiển thị?

URL ảnh phải là đường dẫn trực tiếp đến file ảnh (kết thúc bằng .jpg, .png, .webp). Nhiều trang web không cho phép nhúng ảnh từ domain của họ. Hãy dùng Unsplash hoặc upload lên Google Drive/ImgBB.

Q: Dữ liệu tôi chỉnh sửa có bị mất khi xoá cache không?

Có. Dữ liệu lưu trong localStorage sẽ mất nếu xoá dữ liệu trình duyệt. Hãy sao lưu JSON thường xuyên và cập nhật vào file JS gốc sau mỗi đợt chỉnh sửa lớn.

Q: Làm sao khôi phục dữ liệu đã xoá?

Vào Xuất / Nhập Dữ Liệu → Nhập dữ liệu → chọn file JSON đã sao lưu trước đó. Nếu không có file sao lưu, nhấn Đặt Lại Dữ Liệu Gốc để khôi phục dữ liệu từ file JS.

Q: Tôi có thể thêm tỉnh mới ngoài 8 tỉnh hiện tại?

Có. Mở file admin/index.html, tìm các thẻ <option> trong select tỉnh và thêm tỉnh mới. Đồng thời thêm vào bộ lọc tương ứng trong dia-diem.html và lich-le-hoi.html.
